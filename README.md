# Projeto-Individual
Este projeto consiste no desenvolvimento de um website interativo sobre o anime Demon Slayer – Kimetsu no Yaiba, com o objetivo de apresentar sua história, incentivar novos espectadores e proporcionar interação por meio de quiz e dashboard. O site contará com cadastro e login, formulários dinâmicos, análise das respostas dos usuários e uma página explicativa sobre o anime. O desenvolvimento será realizado utilizando HTML, CSS, JavaScript e Node.js, com conclusão prevista para 24 de novembro de 2025.
